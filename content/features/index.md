## Open Source
The Open Source edition of LocalStack provides the following core Cloud APIs on your local machine.

* **ACM**
* **API Gateway**
* **CloudFormation**
* **CloudWatch**
* **CloudWatch Logs**
* **DynamoDB**
* **DynamoDB Streams**
* **EC2**
* **Elasticsearch Service**
* **EventBridge (CloudWatch Events)**
* **Firehose**
* **IAM**
* **Kinesis**
* **KMS**
* **Lambda**
* **Redshift**
* **Route53**
* **S3**
* **SecretsManager**
* **SES**
* **SNS**
* **SQS**
* **SSM**
* **StepFunctions**
* **STS**
